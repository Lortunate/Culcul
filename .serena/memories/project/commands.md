# Culcul development commands
- Install deps: `flutter pub get`
- Static analysis: `flutter analyze`
- Run tests: `flutter test`
- Launch app: `flutter run`
- Regenerate code after model/router/provider changes: `dart run build_runner build --delete-conflicting-outputs`
- Regenerate i18n after translation changes: `dart run slang`
- Format code: `dart format .`
- Useful Windows shell commands: `Get-ChildItem`, `Get-Content`, `Select-String`, `rg`, `git status --short`, `git diff --stat`.
