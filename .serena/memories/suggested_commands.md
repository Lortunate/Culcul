# Suggested commands (Windows)
- Install deps: `flutter pub get`
- Run app: `flutter run`
- Analyze all: `flutter analyze`
- Analyze one feature: `flutter analyze lib/features/to_view`
- Run tests: `flutter test`
- Format files: `dart format <paths>`
- List files: `dir`, `Get-ChildItem`
- Search text/files: `rg <pattern>`, `rg --files`
