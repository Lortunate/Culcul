# Culcul project overview
- Purpose: Flutter-based third-party Bilibili client.
- Stack: Flutter, Dart, Material UI, Riverpod/Hooks Riverpod, Dio + Retrofit, Freezed/JSON serializable style codegen.
- Structure: `lib/app` app/theme/bootstrap, `lib/features` feature modules, `lib/core` shared core utils/providers, `lib/ui` shared widgets, `lib/data` shared models/repositories, `test` tests.
- Platform context: Windows development environment.
